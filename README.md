# SERS-2SEM-GS-LEE

Os dados utilizados são simulados, porém foram construídos com base em parâmetros extraídos de fontes como ANEEL, INPE, IEA e DOE, garantindo realismo na modelagem do consumo e geração energética.
A solução desenvolvida combina análise de dados energéticos (Opção A) e simulação da adoção de energia renovável (Opção C) para avaliar o consumo e propor melhorias sustentáveis no ambiente de trabalho. Ela utiliza dados simulados de consumo energético diário e modela o impacto da instalação de um sistema de energia solar, estimando desperdícios, economia potencial e redução de consumo líquido.
